Wivivi est un projet open source directement disponible sur la plateforme Github. Wivivi.com a pour vocation d'aider les professionnels du transport et les industriels à faciliter les échanges d'une bourse de fret gratuite ou peu onéreuse.

Langage de programmation

Wivivi est developper via le langage de programmation en Python par le Framework Django. Son environnement serveur est sous Oracle Linux 8 (Centos8) et Nginx pour le Webserver. Html, Css et Js sont mis en avant pour le Frontend et Django et sont Orm pour le Backend.

C'est quoi une bourse de fret open source ?

Le but de l'open source est le partage par la connaissance. Notre travail permet d'affiner et d'améliorer le web généraliste pour d'autres développeurs.
